## In-Class Agenda

### 1. From Discriminative to Generative (10 min)

| Type | Direction | Example |
|------|-----------|---------|
| **Discriminative** | Image → Label | "This is a cat" (classification) |
| **Generative** | Text/Noise → Image | "Generate a photo of a cat on the moon" |

Three main approaches:
- **GANs** (2014) — two networks compete to create realistic images
- **VAEs** (2013) — learn a compressed representation and decode it
- **Diffusion models** (2020–now) — learn to remove noise step by step → state of the art

### 2. GANs — The Idea (15 min)
- **Generator** — creates fake images from random noise
- **Discriminator** — tells real from fake
- They train together: Generator gets better at fooling → Discriminator gets better at detecting → both improve

Analogy: a counterfeiter (Generator) vs. a detective (Discriminator)

Famous GAN outputs: [This Person Does Not Exist](https://thispersondoesnotexist.com/), [GauGAN landscape painting](https://www.nvidia.com/en-us/research/ai-playground/)

```python
# Concept demo — not a real training loop, just the structure
import torch
import torch.nn as nn

# Generator: random noise → image
class Generator(nn.Module):
    def __init__(self, latent_dim=100, img_size=28*28):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(latent_dim, 128),
            nn.ReLU(),
            nn.Linear(128, 256),
            nn.ReLU(),
            nn.Linear(256, img_size),
            nn.Tanh()   # output in [-1, 1]
        )
    def forward(self, z):
        return self.net(z)

# Discriminator: image → real/fake probability
class Discriminator(nn.Module):
    def __init__(self, img_size=28*28):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(img_size, 256),
            nn.LeakyReLU(0.2),
            nn.Linear(256, 64),
            nn.LeakyReLU(0.2),
            nn.Linear(64, 1),
            nn.Sigmoid()   # output in [0, 1]
        )
    def forward(self, img):
        return self.net(img)
```

### 3. Diffusion Models — The Modern Approach (20 min)
- Training: take a real image → add noise step by step → teach the model to reverse this
- Generation: start from pure noise → remove noise step by step → get a real image
- Text conditioning: the text prompt guides which direction to remove noise

```python
# Using Hugging Face Diffusers — the practical way
from diffusers import StableDiffusionPipeline
import torch

# Load Stable Diffusion (requires ~4GB GPU RAM or use Colab)
pipe = StableDiffusionPipeline.from_pretrained(
    "runwayml/stable-diffusion-v1-5",
    torch_dtype=torch.float16
)
pipe = pipe.to("cuda")   # use GPU

# Generate an image from a text prompt
image = pipe(
    prompt="a red apple on a wooden table, photorealistic, 4k",
    negative_prompt="blurry, low quality, cartoon",
    num_inference_steps=30,
    guidance_scale=7.5
).images[0]

image.save("generated_apple.png")
image  # displays in Jupyter/Colab
```

Key parameters:
| Parameter | Effect |
|-----------|--------|
| `prompt` | What you want in the image |
| `negative_prompt` | What you do NOT want |
| `num_inference_steps` | More steps = higher quality (slower) |
| `guidance_scale` | Higher = follows prompt more strictly (7–12 is typical) |
| `seed` | Fix for reproducible results |

### 5. Prompt Engineering for Images (5 min)
Prompts for image generation are different from text prompts:
- **Style words matter:** "photorealistic", "oil painting", "watercolor", "digital art", "4k", "cinematic"
- **Composition words:** "close-up", "wide angle", "bird's eye view", "bokeh background"
- **Negative prompts remove problems:** "blurry, low quality, cartoon, deformed, ugly, watermark"
- **Artist references:** "in the style of Van Gogh", "Studio Ghibli style"
