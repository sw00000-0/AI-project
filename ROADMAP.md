## Phase Breakdown

### Phase 1 — Foundation (Sessions 1–3)
Learn the core Python tools every AI project uses. No deep math required — focus on reading, manipulating, and visualising data.

**Skills unlocked:** Load data, clean it, explore it, plot it, run a basic ML model.

---

### Phase 2 — Core AI (Sessions 4–5)
Learn to use pre-built AI models through Hugging Face and APIs. Build your first chatbot UI.

**Skills unlocked:** Use LLMs via APIs, build interactive apps with Streamlit.

---

### Phase 3 — AI for Images (Sessions 6–9)
Understand how AI works with images — from basic processing to generating new ones.

**Skills unlocked:** Process images, use CNNs, generate images, combine text + image AI.

---

### Phase 4 — Project (Sessions 10–14)
Apply everything to build, document, and present your own AI project.

**Skills unlocked:** Research, design, implement, evaluate, and present a real AI application.
